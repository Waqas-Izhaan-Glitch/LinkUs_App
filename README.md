# LinkUs App

> A command-line social media application built in C++17 — post, connect, and engage. No database, no framework. Just clean C++.

**Created by Waqas Izhaan**

---

## What is LinkUs?

LinkUs is a fully functional social media app that runs in your terminal. You can create accounts, post updates, follow other users, like posts, search content, and view a personalized news feed — all from the command line. Data is automatically saved to a file so nothing is lost between sessions.

---

## Features

- Create Account — register with a username and bio
- Post — share updates up to 280 characters
- Follow / Unfollow — build your own network
- News Feed — see the latest posts from people you follow
- Like / Unlike — toggle likes on any post
- Search — find posts by keyword, case-insensitive
- View Profile — see stats, followers, following, and recent posts
- Delete Post — remove your own posts
- File Persistence — all data saved automatically on exit

## Getting Started

### Requirements
- g++ compiler with C++17 support
- Terminal or Command Prompt

## How to Use

```
========== LINKUS APP ==========
1. Create User
2. Post
3. Follow User
4. Unfollow User
5. View Feed
6. Like Post
7. Search Posts
8. View Profile
9. Show All Users
10. Delete Post
0. Exit
```

Quick start in 3 steps:
1. Choose `1` to create your account — enter a username and bio
2. Choose `2` to post — enter your user ID and write your post
3. Choose `5` to view your personalized feed
